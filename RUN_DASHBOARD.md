# Run the Live Dashboard

## Quick Start (60 seconds)

### Step 1: Start the Server

```bash
cd C:\Users\Administrator\Desktop\ray-ai-case-study
python server.py
```

You should see:
```
============================================================
Ärztehaus Radar Dashboard Server
============================================================

Starting server on http://localhost:5000
Open in browser: http://localhost:5000

API endpoint: http://localhost:5000/api/leads

Press Ctrl+C to stop
============================================================
```

### Step 2: Open Dashboard in Browser

Go to:
```
http://localhost:5000
```

The live dashboard will load with all 10 demo projects, organized by quality (high/medium/low).

---

## How It Works

**Server**: Reads `output/leads.csv` and serves it as JSON
**Dashboard**: Fetches the JSON and renders it beautifully

## File Structure

```
server.py                    ← Flask server (reads CSV, serves JSON)
output/
├── dashboard-live.html     ← Live dashboard (reads from API)
├── leads.csv               ← Data source (auto-generated by main.py)
└── raw_projects.json       ← Raw project data
```

## API Endpoint

While the server is running, you can also query the API directly:

```bash
# In PowerShell:
curl http://localhost:5000/api/leads | ConvertFrom-Json | jq .

# Or in browser:
http://localhost:5000/api/leads
```

Returns all projects as JSON with stats.

## Troubleshooting

### Port 5000 Already in Use
```bash
# Find what's using port 5000
netstat -ano | findstr :5000

# Kill it (replace PID with the number)
taskkill /PID <PID> /F

# OR change port in server.py (line 34):
# app.run(debug=True, port=5001)
```

### "ModuleNotFoundError: No module named 'flask'"
```bash
pip install flask
```

### Dashboard Shows "Error Loading Data"
1. Make sure `output/leads.csv` exists
2. Run `python main.py` to generate it
3. Restart server: Press Ctrl+C, then `python server.py`
